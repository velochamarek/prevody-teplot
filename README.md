# prevody-teplot
Jednoduchá webová aplikace pro převod teplot mezi:
- Celsius
- Fahrenheit
- Kelvin

## Použité jazyky
- HTML
- CSS
- JavaScript

## Funkce
- Převod mezi všemi třemi jednotkami
- Ošetření neplatného vstupu
- Jednoduché uživatelské rozhraní
